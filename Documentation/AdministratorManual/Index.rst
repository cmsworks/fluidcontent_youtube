Administrator Manual
====================

Installation
------------

The Fluid Content YouTube extension can be installed from the TER (TYPO3 Extension Repository):

#. Install the extension from the Extension Manager
#. Include the static template

This extension depends on ``EXT:fluidcontent`` and ``EXT:vhs``, so they will be installed too.

Configuration
-------------

The extension can be configured using TypoScript. See the :ref:`config-reference`.
